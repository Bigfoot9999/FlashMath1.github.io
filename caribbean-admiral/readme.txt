how to add this game to your blog?

Adding the game is easy! Here are some instructions to help you.

1) Open your caribbeanadmiral.html inside a notepad (or other editor) and copy paste the embed code to your blog or website.
2) Upload the file caribbeanadmiral.swf to your server on the same remote path as the page displaying the game.
3) Have fun!
